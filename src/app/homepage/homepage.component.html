Kneelawk's Website

Welcome to my website. This page is still under construction, but at least it says something now.

I hope to add more things to this site in the future. I have several screen-saver programs that I think would do well as website backgrounds. Maybe I'll have a button to switch between them. I also think it might be interesting to build an online version of my Rust-based fractal generator. However, I don't have a lot of time to work on projects like this at the moment, so updates will happen when they happen.

This website is written using the Angular framework and hosted using Github-Pages. Angular is definitely over-kill for a site like this, but using it here provides a fun learning opportunity, while allowing me to experiment with deploying Webpack-based applications to a Github-Pages-based platform.

If you would like to have a look at the source code for this website, feel free to check out the repository on my Github.
